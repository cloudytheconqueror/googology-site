Eighty three
83

The 23rd prime (between 79 and 89).

83 is the number of right-truncatable primes in base 10. A right-truncatable prime is a prime that is one digit long or can be turned into another right-truncatable prime by removing the rightmost digit. The largest is 73,939,133 (see that entry for more).

Scott Aaronson's essay "Who Can Name the Bigger Number?" starts with the following paragraph:

In an old joke, two noblemen vie to name the bigger number. The first, after ruminating for hours, triumphantly announces "Eighty-three!" The second, mightily impressed, replies "You win."

Of course, in later parts of this list, we'll introduce numbers that make the nobleman's 83 look tiny.

One hundred sixty three
163

163 is the 37th prime number.

Previously, people misunderstood the definition of mixed factorial (see 6561) as 1+(2*(3^(...((x-1){x-3}x)...))), when it should be (...((((1+2)*3)^4)^^5)...){x-3}x. Using the misunderstood definition, tetrafact or 4* evaluates to 1+(2*(3^4)) = 1+(2*81) = 1+162 = 163, while using the correct definition, 4* = ((1+2)*3)^4 = (3*3)^4 = 9^4 = 6561.

Six thousand five hundred sixty one / Tetrafact
4* = 9^4 = 6561

Tetrafact is a googolism defined using SpongeTechX's mixed factorial (SpongeTechX is a GWiki user), which is defined as x* = (...((((1+2)*3)^4)^^5)...){x-3}x (note the left-to-right order; see 163). Tetrafact is 4* = ((1+2)*3)^4 = (3*3)^4 = 9^4 = 6561. It is the ninth tesseract number (9^4) and 3 to the 8th power (3^8).

Tetrafact is not to be confused with tethrafact, which is E100(#^^#)^#100 in extended cascading-E (a much, much larger number).

Alpha
[A] = 12,549,900

The number "Alpha" given in Sbiis Saibian's Jacob's Ladder article. In the article, the number n is notated as (AAAAA...AAAAA) with n A's. The bulk of the article contains a "massive" number, which is (AAAAA...AAAAA) with 167,332 A's. Then Saibian defined Alpha as 167,332 lines of A's, each containing 75 A's, with the entire thing contained between parentheses. The result is a number with 167,332*75 = 12,549,900 A's between parentheses.

Saibian decided to call the number with Alpha lines of 75 A's between ()'s Beta, and the number with Beta lines is called Gamma, and Delta and Epsilon are created in the same way. Then he devised a shorthand for these: Alpha = [A] (note the square brackets), Beta = [AA] = 941,242,500, Gamma = [AAA] = 70,593,187,500 etc.

This is followed by many extensions, which began by putting Alpha A's between square brackets and giving it the notation [[A]], and then putting that many A's between square brackets and calling it [[[A]]] and so on. Then Saibian created a sequence where the first term is [A], and the terms after that are defined as [...[A]...] with the number of pairs of brackets equal to the previous term, and it gets worse from there.

Ten trigintillion
10^94

This is the number of volumes of Wolfgang H. Nitsche's multi-volume book Googolplex Written Out, which includes the entire decimal expansion of googolplex.

Each volume consists of 405 pages, of with 400 pages actually containing the digits. Each digit-containing page includes 2500 zeroes, with the exception of page 5 which has 20 less zeroes, and page 404 which has 20 more. Therefore each volume includes 1,000,000 zeroes, so there are a googol zeroes total in the entire 10^94 volume book. Page 5 of the first volume also contains the digit 1, the first digit of googolplex.

23rd Mersenne prime
2^44,497-1
~ 8.545098*10^13,394

This was the first gigantic prime (primes with 10,000 or more digits) discovered. It was found by Nelson and Slowinski in 1979.

Curious fact: The first three Mersenne prime exponents that start with 4 are 4253, 4423 and 44,497. 4253 starts with one 4, 4423 starts with two 4's, and 44,497 starts with three 4's. However, the next exponent that start with 4, 42,643,801, doesn't follow that pattern.

F33 = 2^2^33+1
~9.630350*10^2,585,827,972

This Fermat number is the smallest Fermat number that's not known to be either prime or composite. Considering it has over two billion digits, it is currently too large to test for primality. It became the smallest Fermat number with unknown primality when a factor of F31 was found in April 2001.

This number is equal to one plus googolpleiv, since 4^4^16 = 2^(2*4^16) = 2^(2*2^32) = 2^2^33.

Grangol-carta-gralgathor
E100#^#^##100#100

Here we encounter the -carta- infix. An infix, analogous to a prefix and a suffix, is a root that's put in-between words.
If a googolism X is represented as E%, and Y as @100, then X-carta-Y is defined as @%. X-carta-Y, "carta" meaning "map" in Greek, can be thought as "mapping" X to Y. In this example, X is grangol = E100#100, Y is gralgathor = E100#^#^##100, so @ = E100#^#^## and % = 100#100, resulting in grangol-carta-gralgathor = E100#^#^##100#100.

The -carta- infix was first introduced in 2014 when xE^ was released, then later retroactively applied to earlier Saibianisms (like this one) when the list of number names was greatly expanded.
